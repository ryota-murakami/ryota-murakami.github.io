Flow
Flow is a static type checker and is not a static Type language, code does move even if it is a Type error, it has a unique feature that it gives Type information to JavaScript Dynamic Type Language and makes it possible to analyze it.
Although I am not familiar with the history of the programming language, the initial Type is information for the compiler, such as instructing the necessary allocate of memory, compiling at a high speed by describing argument and return type in advance, etc. It seems TypeSystem role was supported compiler.
However, such as remarkable improvement of machine power, high performance of IDE, Emergence of programming style effectively using Type such as Gof design pattern, these were changed in the situation.
Type has more information support to humans rather than a compiler.

A type system, that has only responsibility as annotation information that does not participate in the execution result of the code but assists humans.
